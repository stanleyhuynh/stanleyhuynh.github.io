### Speed Run
I am really finding these runs quite fun in terms of how different it is from the normal jog/run style I usually do.

## Hills Run
For this run, I went into it not knowing of any hills.  I enjoyed researching places, and ways to find hills for these types of runs.  I ended up utilizing the Strava app to find run segments from my community which had a convenient elevation stat attached to each run segment.  I ended up finding one nearby less than 10 minutes jog away which was great and the workout was quite enjoyable since as defined only had a short volume of running time.

## Speed Run
I thought this run was quite hard since its supposed to be kept at a faster pace than usual to test my ability to be uncomfortable.  I was aiming for a 5:30/km pace but according to the Nike Run Club app I fluctuated often but averaged 5:45/km.  My takeaway for this run would be to try to push myself more.  During the run I felt like I had to keep a pace that would allow me to complete the tempo run, but after the run I felt like I could have pushed myself more as I did not feel that sore.

### Long Run
This was by far the longest run I've ever done which is great a Personal Record and hope to achieve more. The pace was also an improvement from my previous runs I was floating just above 6:00/km which gives me hope to at least break sub 1-hour 10km run but I will aim for more as originally planned.  I didn't think I'd enjoy these runs but after this run I am starting to think differently.  Running these longer runs, listening to podcasts can be quite therapeutic.
